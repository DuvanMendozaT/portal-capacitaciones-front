# Portal de Capacitaciones Interactivo - Frontend

Aplicación frontend del proyecto **Portal de Capacitaciones Interactivo**. La aplicación permite a los usuarios autenticarse, consultar cursos organizados por módulos, realizar seguimiento a su progreso y visualizar insignias e histórico de aprendizaje.

## Objetivo

Construir una interfaz web simple, navegable y funcional para consumir los servicios del backend del portal de capacitaciones y ofrecer una experiencia clara para estudiantes y administradores.

## Funcionalidades

- Login básico
- Consulta de capacitaciones por módulos:
  - Fullstack
  - APIs e Integraciones
  - Cloud
  - Data Engineer
- Visualización del detalle de cursos
- Marcar cursos como iniciados o completados
- Visualización de insignias generadas
- Consulta de histórico de cursos
- Módulo administrativo básico para agregar y editar cursos
- Navegación sencilla y diseño responsivo

## Tecnologías utilizadas

- Angular
- TypeScript
- HTML5
- CSS3
- RxJS
- Angular Router
- Reactive Forms
